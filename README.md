# 캠퍼스타운 허브 (CampusTown Hub)

숙명 스타트업 캠퍼스 입점기업·거래처·기관·교수·학생을 위한 **회의 운영 사이트**입니다.

여러 회의를 한곳에서 관리하고, 발표·화상회의·녹취·회의록을 하나의 화면에서 진행합니다.

---

## 주요 기능

| 기능 | 설명 |
|---|---|
| **회원가입·로그인** | 본인 이메일 또는 소셜 계정(Google·Naver·Kakao·Apple) |
| **역할 관리** | 운영자(사업자) · 개최자 · 초대자 |
| **회의 검색** | 타이핑 또는 **음성**으로 검색 ("오늘 10시 회의", 참석자 이름) |
| **발표 슬라이드** | 첫 화면에서 **자동 발표** — 음성(TTS) + 자막 + 용어 설명 (약 3분) |
| **화상회의** | Jitsi Meet 임베드 — **무료·무제한·계정 불필요** (Zoom 40분 제한 없음) |
| **실시간 녹취** | 브라우저 음성인식(STT)으로 텍스트 변환 |
| **회의록 작성** | 녹취를 정리해 회의록·이메일 초안 생성 |
| **녹취 자동 파기** | 회의 종료 시 **녹취 원문 즉시 삭제**, 회의록·정리 스크립트만 보존 |
| **잡(Job) 연결** | 회의에서 나온 제안을 다음 회의로 이어가기 |
| **백업** | JSON 내보내기·가져오기 (기기 간 이동) |

---

## 개인정보 보호

- **음성 파일은 만들어지지 않습니다.** 브라우저 음성인식은 소리를 실시간으로 텍스트로만 변환합니다.
- **회의 종료 시 녹취 원문이 자동 파기됩니다.** 정리된 스크립트와 회의록만 남고, 파기 시각이 기록됩니다.
- 민감한 회의라면 회의록을 이메일로 내보낸 뒤 기록을 삭제하시길 권합니다.

## 사용 방법

### 접속
GitHub Pages 주소로 접속합니다.

```
https://<사용자명>.github.io/campustown-hub/
```

### 회의 참여
1. **회원가입** — 본인 이메일 또는 소셜 계정
2. **로그인** — 초대된 회의가 목록에 표시됩니다
3. **입장** — 슬라이드·화상회의·녹취를 진행합니다

### 회의 만들기 (개최자·운영자)
1. 회의 목록에서 **+ 새 회의 만들기**
2. **사전 준비** 탭에서 참석자 이메일 등록
3. **화상회의** 탭에서 링크를 복사하거나 이메일로 초대

## 설정 (관리자용)

### Supabase 연결

여러 사람이 **같은 회의·회의록을 공유**하려면 Supabase(무료)가 필요합니다.

`index.html` 상단에서:

```javascript
const SUPABASE_URL = "https://xxxxx.supabase.co";
const SUPABASE_ANON_KEY = "eyJhbGci...";
```

비워두면 이 브라우저에만 저장하는 **로컬 모드**로 동작합니다.

자세한 설정 방법은 [SETUP.md](SETUP.md)를 참고하세요.

### 가입 정책

```javascript
let OPEN_SIGNUP = true;   // 누구나 본인 이메일로 가입
let OPEN_SIGNUP = false;  // 초대된 이메일만 가입 (기밀 회의용)
```

## 기술

- 단일 HTML 파일 (프레임워크 없음)
- Supabase (PostgreSQL) — 데이터 공유
- Jitsi Meet — 화상회의
- Web Speech API — 음성인식·음성합성
- GitHub Pages — 호스팅

### 확장 (MCP·로봇 연동)

`window.MedITHub` 액션 레이어로 외부 시스템과 연결할 수 있습니다.

```javascript
MedITHub.searchMeetings("오늘 10시 회의")
MedITHub.getMeetingStatus(id)     // 진행 상황
MedITHub.getLiveTranscript()      // 실시간 녹취
MedITHub.listJobs()
```
